# 0xVM

### Architecture
32-Bit architecture

### Sizes
`WORD` = 4-Byte <br>
Register-Address = 1-Byte <br>
Memory-Address = 4-Byte

### Registers
`acc`: Contains the result of the last arithmetic operation <br>
`pc`: program counter <br>
`sr`: status register <br>
`r[1-8]`: 8 general purpose registers
